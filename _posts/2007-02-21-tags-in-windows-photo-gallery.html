Heres a little tip when adding tags in
Windows Photo Gallery in Vista.

If you want to make the tags groupable, eg list places together, then you can do so
by formatting the tag as follows:

Places/Reading/Reading Festival;

On the left handside you will see the treeview contain subnodes

You can also drag & drop the nodes, which updates any photos with the tags.
